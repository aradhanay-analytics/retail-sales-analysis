# Retail Sales Analysis

## Objective
To analyze retail sales data and identify key business insights such as regional performance, product trends, and seasonal patterns.

## Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard Visualization)

## Key Insights
- West region generated highest overall sales
- Technology category contributed maximum revenue
- Sales peak observed during November–December
- Top-performing products drive majority of revenue
